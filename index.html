Element Selector

Selects all elements of the specified type.

p { color: blue; }
div { padding: 10px; }
Class Selector

Selects elements with a specific class attribute.

.highlight { background: yellow; }
.button { padding: 10px 20px; }
ID Selector

Selects a single element with a specific id attribute.

#header { height: 100px; }
#main-content { width: 80%; }
Descendant Combinator

Selects elements that are descendants of another element.

div p { color: red; }
.container .item { margin: 10px; }
Child Combinator

Selects direct children of an element.

ul > li { list-style: square; }
div > p { font-weight: bold; }
Adjacent Sibling

Selects an element that is immediately after another element.

h2 + p { margin-top: 0; }
.heading + .content { padding: 10px; }
:hover Pseudo-class

Selects an element when mouse over it.

a:hover { color: red; }
.button:hover { background: #333; }
:nth-child Pseudo-class

Selects elements based on their position.

li:nth-child(odd) { background: #eee; }
tr:nth-child(2n) { background: #f5f5f5; }
:not Pseudo-class

Selects elements that do not match a selector.

div:not(.container) { border: 1px solid; }
p:not(:first-child) { margin-top: 10px; }
